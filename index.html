Instructions - to interact with Physics_JS please enter a number for 'number of particles' and 'mass of particles' and press ENTER. Deatils regarding each input can be found by hover your mouse over the text field. The lights that you see are collision detection. You can also give a numbers for both the width and height of the canvas. Use the slider to adjust the strength of gravity. At anytime you can pause, clear the canvas, or add a central attractor.

  • Number of particles

    Enter a number between 1 and 200 to see the number of particle interactions

  • Mass of particles

    Enter a number between 1 and 500 to assign mass to each particle

  • Set width of canvas

    Enter a number between 200 and 800 to set the width of the canvas

  • Set height of canvas

    Enter a number between 200 and 500 to set the height of the canvas
